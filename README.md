# 1. 使用准备
## 1.1 添加依赖
```xml
<dependency>
    <groupId>top.rows.cloud.owl.job</groupId>
    <artifactId>owl-job-core</artifactId>
    <version>1.0.0</version>
</dependency>
```
## 1.2  其他依赖
### Springboot 项目
```xml
<dependency>
    <groupId>top.rows.cloud.owl.job</groupId>
    <artifactId>owl-job-spring-boot-starter</artifactId>
    <version>1.0.0</version>
</dependency>
```
